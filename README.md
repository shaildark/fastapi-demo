
Create Virtual Environment

```python -m venv .venv```

Activate Virtual Environment

```.venv\Scripts\activate```

Deactivate Virtual Environment

```deactivate```

Upgrade pip package manager
```python -m pip install --upgrade pip```

install FastAPI

```pip install "fastapi[standard]"```

Run FastAPI

```fastapi dev main.py```
